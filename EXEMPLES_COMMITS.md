# 📝 Exemples de Commits - Copier-Coller

## 📌 Comment utiliser ce fichier

1. Trouve le type de commit que tu veux faire
2. Copie le texte du **Summary** et **Description**
3. Colle-le dans GitHub Desktop
4. Modifie les détails si nécessaire

---

## ✨ FEATURE - Nouvelle Fonctionnalité

### Ajout d'une page

**Summary** :
```
✨ [Feature] Ajout de la page [NomDeLaPage]
```

**Description** :
```
Description détaillée :
- Création de [NomDeLaPage].xaml et .xaml.cs
- Ajout de [fonctionnalité 1]
- Ajout de [fonctionnalité 2]
- Intégration avec [DataManager/autre système]

Fichiers ajoutés :
- Pages/[NomDeLaPage].xaml
- Pages/[NomDeLaPage].xaml.cs

Impact : Ajout d'une nouvelle page fonctionnelle
```

## 🎯 CONSEILS

### Bon message de commit

✅ **BON** :
```
Summary: ✨ [Feature] Ajout de la page Factures
Description: Détails précis avec tirets
```

❌ **MAUVAIS** :
```
Summary: Update
Description: (vide)
```

---

### Longueur idéale

- **Summary** : 50 caractères maximum
- **Description** : Autant que nécessaire (mais structuré)
